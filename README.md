# Pomodoro Clock

This Pomodoro Clock was built with React during the PERSEVERE Coding Bootcamp while I was incarcerated.

The project marked an important milestone in my development as I transitioned from static web pages to interactive React applications. It demonstrates component-based development, state management, event handling, and user interaction.

The application has been preserved in its original form to document my growth as a software developer.

## Technologies Used

- React
- JavaScript
- HTML5
- CSS3
- Bootstrap

## Features

- Adjustable work session length
- Adjustable break length
- Countdown timer
- Start and reset controls
- Responsive user interface
- Dynamic state updates

## What I Learned

Through this project I gained experience with:

- React components
- React Hooks
- State management with `useState`
- Event handling
- Timer logic
- Building interactive user interfaces
- Structuring larger React applications

## Reflection

This project represents my early experience building applications with React. While modern development tools report a few lint warnings due to changes in the React ecosystem, the application has been preserved in its original form to accurately document my learning process. The dependency files have been refreshed so the project can still be installed and run in a modern development environment.
